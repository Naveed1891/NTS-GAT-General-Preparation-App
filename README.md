# 📚 NTS GAT General Preparation App

![React](https://img.shields.io/badge/Frontend-React-blue)
![AI Powered](https://img.shields.io/badge/AI-Powered-green)
![License](https://img.shields.io/badge/License-MIT-yellow)
![Status](https://img.shields.io/badge/Project-Active-brightgreen)

An **AI-powered web application** built with **React** to help students prepare effectively for the **NTS GAT General exam**.  
The platform provides **AI-generated quizzes, mock tests, an AI Tutor, and progress analytics** to support smart and focused exam preparation.

---

# ✨ Features

## 🧠 AI-Generated Quizzes
- Dynamic quizzes generated using AI
- Questions aligned with the **official NTS GAT General syllabus**
- Maintains **variety and exam-level difficulty**

## 📝 Mock Tests
- Full-length mock exams
- Designed to simulate the **real test environment**
- Helps students improve **speed and accuracy**

## 🤖 AI Tutor
- Interactive AI assistant for learning
- Students can ask questions and receive **concept explanations**
- Helps clarify difficult topics instantly

## 📊 Student Dashboard
- Tracks student progress
- Displays quiz and mock test performance
- Helps students monitor their preparation

## 📈 Performance Analytics
- Analyzes quiz and test results
- Identifies **weak topics and learning gaps**
- Helps students focus on areas that need improvement

---

# 🎯 Purpose

Preparing for the **GAT General exam** requires consistent practice and concept clarity.  
This application combines **AI-driven assessments, personalized feedback, and analytics** to create an **efficient and interactive preparation platform for students**.

# ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### 2️⃣ Navigate to the project directory

```bash
cd your-repository-name
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```
